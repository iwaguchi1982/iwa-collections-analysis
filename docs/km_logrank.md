# KM曲線 / log-rank検定

## Kaplan–Meier（KM）

KMは、時間経過に伴う生存確率を推定し、群間の差を可視化します。

- 曲線が早く落ちるほど、イベント（死亡）が早い傾向
- 信頼区間（CI）が広い場合はデータが少ない/不安定

### 注意点（初心者が陥りやすい罠）
- **「交差」に注意**: グラフの線が途中で交差している場合、時間によってリスクが逆転しています（比例ハザード性が無い状態）。この場合、出力される単一の p値 や Cox ハザード比を単純に信用してはいけません。
- **n数が少ない**: 各群の N が小さすぎると、たった1人の死亡で曲線が大きく階段状に落ち、不十分なデータから誤った結論を導きやすくなります（Guardrails参照）。
- 検閲（ひげマーク）が多いと後半の推定が不安定になりがちです。後半の急激な低下を「効果がある！」と過大評価しないようにしてください。

## log-rank検定

KM曲線の群間差を検定します。

- 2群：High vs Low
- 4群：HiHi / HiLo / LoHi / LoLo

### よくある誤解
- **p < 0.05 ＝ 効果が絶大である** とは限らない。（サンプルサイズが巨大なら、ごく微小な差でも有意になります。必ず効果量＝HR/ハザード比 も一緒に見ること）
- **p < 0.05 ＝ 因果関係がある** ではない。（別の交絡因子、例えば年齢やステージの偏りが生み出した見かけの差かもしれません）
- **一番分離する cutoff の p値を探す** のは危険。（「いろいろ試して一番良かったp値」は多重検定のペナルティを受け、偶然のノイズ（偽陽性）である確率が非常に高いです。発見したしきい値は外部データで検証する必要があります）


## Restricted Mean Survival Time (RMST)

RMST（制限付き平均生存時間）は、指定された期間（例：60ヶ月）までのKM曲線の下の面積を計算し、その期間内での「平均生存期間（月数）」を推定する手法です。

- **比例ハザード性が崩れる場合（曲線が途中で交差する等）** に、Cox回帰のハザード比（HR）よりも客観的で解釈しやすい効果量となります。
- 最近のトップジャーナル（特に免疫チェックポイント阻害剤などの遅効性効果が見込まれる分野）で強く推奨されています。

### 解釈と注意点
- **絶対的な効果量**: 異なる群間のRMSTの差（Difference in Months）は、「その指定期間内で、平均して何ヶ月生存期間が延長/短縮されたか」という非常に直感的な臨床的インパクト（Effect Size）を表します。
- **p値に関する注釈**: 現在の `iwa-collections-analysis` では、「各群のRMSTの絶対値（面積）」と「群間の差（Difference in Months）」を提示する形で表現しています。この「群間の差」に対する正確なp値（漸近分散などを用いた検定）を一発で計算・検定する仕組みは現在組み込まれていません。そのため、p値ではなく「生存期間の延長（絶対月数）」という**効果量の指標**として結果を解釈してください。
