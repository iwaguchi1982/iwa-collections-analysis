# 🧪 Validation (Cross-Cohort Validation) モード

本機能は、あるデータセットで見つけた「しきい値のルール」が、独立した別のデータセット（外部コホート）でも**再現するか**を検証するための専用モードです。

## 目的と背景
生存時間解析における最も多い失敗は、**「手元のデータセット（特定の集団）に偶然当てはまっただけのしきい値を、普遍的な真理だと錯覚してしまうこと（過学習: Overfitting）」**です。
これを防ぐためには、探索的解析（Discovery）で見つけたルールを完全に固定し、測定プラットフォームや患者層が異なる別のデータセット（Validation）に当てはめて、同じように有意差（p < 0.05）が出るかを確認するプロセスが不可欠です。

---

## 使い方
1. **データセットの選択**: 
   - `Discovery Dataset` (探索用) と `Validation Dataset` (検証用) をそれぞれ選びます。
2. **ルールの固定**: 
   - 検証したい対象の `Gene Symbol` を入力し、Discovery 側で最も良く分かれた「しきい値（Percentile または Z-score）」を指定します。
3. **実行 (`Run Validation`)**: 
   - Discovery 側と Validation 側で同時に計算が走り、左右に Kaplan-Meier 曲線と Cox 回帰のハザード比（Unadjusted）が表示されます。

---

## 計算基準: バッチエフェクトの吸収 (Transfer Method)
生の発現量（例: Illumina RNA-Seq の絶対値 500）は、異なる測定プラットフォーム（例: マイクロアレイや別施設のRNA-Seq）には直接そのまま適用できません（バッチエフェクト）。
本アプリでは、このスケールの違いを吸収し、純粋に「集団内での相対的な位置づけ」をルールとして転送するために、以下の2つの手法を提供しています。

### 1. Percentile 転送 (順位ベース)
**「全体のうち上から何%の層か」**というルールを転送します。
- Discovery 側で「Thr=50 (上位50%)」を指定した場合：
  - Discovery データの全体の中央値が $c_1$ ならば、「発現量 >= $c_1$ の人を High」
  - Validation データの全体の中央値が $c_2$ ならば、「発現量 >= $c_2$ の人を High」
- **メリット**: 分布の形が違っていても、群の割合（N数の比率）が維持されるため直感的に分かりやすい。実務や多くの論文で使われる標準的な手法です。

### 2. Z-score 転送 (標準化ベース)
**「集団の平均値から相対的にどれくらい離れているか」**というルールを転送します。
- Discovery 側で「Thr=1.0 (Z-score > 1.0)」を指定した場合：
  - 各データセット内で、「(個人の発現量 - その集団の平均発現量) / その集団の標準偏差」として Z-score を独立して計算します。
  - そして、両方の集団において純粋に「Z-score >= 1.0」の人を High 群として判定します。
- **メリット**: 「上位◯%」と人数で区切るのではなく、純粋な生物学的バラツキ（発現の強さの分布）に依存して区切るため、コホート間で極端な変異をもつ患者の割合が異なる場合等に威力を発揮します。

## 解釈の注意点 (Interpretation)

結果の解釈には、左右のコホートでの p 値（Log-rank 等）を比較します。

- **再現成功 💮 (p_disc < 0.05 かつ p_val < 0.05)**
  - その遺伝子のしきい値ルールは、データセット固有のノイズを超えて強い予後予測能力（頑健性）を持っています。論文のメイン Figure となり得る結果です。
- **再現失敗 ⚠️ (p_disc < 0.05 かつ p_val >= 0.05)**
  - Discovery 側での有意差は、**過学習 (Overfitting) に過ぎなかった可能性**が濃厚です。あるいは、Discovery と Validation 間で「治療歴」「がんのサブタイプ構成」「人種」といったベースラインの生物学的前提が決定的に異なっている（交絡因子が隠れている）可能性があります。
- **Discovery で非有意 (p_disc >= 0.05)**
  - そもそも探索用データセットにおいて、その条件での仮説付けが成立していません。アプリ上は Info メッセージが表示されます。
